# ClientLog - Scripts de Automatización PowerShell

Este directorio contiene scripts PowerShell para automatizar la creación de contenido en ClientLog.

## Scripts Disponibles

### 1. nuevo-cliente.ps1
Crea una nueva carpeta de cliente con toda la estructura necesaria.

**Uso:**
```powershell
.\scripts\nuevo-cliente.ps1
```

### 2. nueva-incidencia.ps1
Crea una nueva incidencia para un cliente existente.

**Uso:**
```powershell
.\scripts\nueva-incidencia.ps1
```

### 3. nueva-actividad.ps1
Crea una nueva actividad comercial (venta, oferta, oportunidad).

**Uso:**
```powershell
.\scripts\nueva-actividad.ps1
```

### 4. nueva-reunion.ps1
Registra una nueva reunión con un cliente.

**Uso:**
```powershell
.\scripts\nueva-reunion.ps1
```

### 5. start.ps1
Inicia el servidor Jekyll con livereload.

**Uso:**
```powershell
.\start.ps1
```

## Requisitos

- PowerShell 5.1 o superior
- Jekyll instalado
- Ruby instalado

## Notas

Todos los scripts son interactivos y solicitarán la información necesaria paso a paso.
